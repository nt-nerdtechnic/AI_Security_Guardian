# 貢獻者規範 (Contributor Covenant Code of Conduct)

## 我們的承諾

為了營造一個開放且友好的環境，我們作為貢獻者和維護者，承諾不分年齡、體型、殘疾、族裔、性別特徵、性別認同及表達、經驗水準、教育程度、社會經濟地位、國籍、外貌、種族、宗教、或性取向，讓參與本專案及社群的所有人都能免於騷擾。

## 我們的標準

有助於為社群營造正面環境的行為範例包括：

* 使用包容及得體的語言
* 尊重不同的觀點與經驗
* 體面地接受具建設性的批評
* 關注對社群最有利的事情
* 對其他社群成員展現同理心

參與者不可接受的行為範例包括：

* 性挑逗或淫穢語言，以及不恰當的性示好或性挑釁行為
* 挑釁、羞辱或貶損的言論，以及人身或政治攻擊
* 公開或私下的騷擾
* 未經明確許可就發佈他人的私人資訊（如物理地址或電子郵件地址）
* 其他在專業環境中可以被合理認為是不適當的行為

## 我們的職責

專案維護者有責任闡明可接受行為的標準，並將針對任何不當行為採取適當、公平的糾正措施。

專案維護者有權利和責任刪除、編輯或拒絕不符合本行為準則的評論（Comments）、提交（Commits）、程式碼、維基（Wiki）編輯、問題（Issues）和其他貢獻，並對任何表現出其認為不適當、威脅、冒犯或有害的其他行為的貢獻者進行臨時或永久性的禁言。

## 適用範圍

當個人代表本專案或其社群時，本行為準則適用於專案空間及公共空間。代表專案或社群的示例包括使用官方專案電子郵件地址、透過官方社群媒體帳號發布訊息，或在線上或線下活動中擔任指定代表。專案維護者可以進一步定義和澄清專案的代表性。

## 執行細節

可透過 [填寫聯絡方式] 聯繫專案團隊來舉報虐待、騷擾或其他不可接受的行為。所有投訴都將得到審理和調查，並將產生被認為必要且適合該情況的答覆。專案團隊有義務對舉報人保持保密。關於具體執行政策的更多細節可能會單獨發布。

不真誠地遵守或執行行為準則的專案維護者，可能會面臨由專案領導層其他成員確定的臨時或永久性處分。
